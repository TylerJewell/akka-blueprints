Agentic AI Blueprints

A library of agentic AI patterns, each one a runnable system. Every blueprint is a natural-language brief that Akka Specify turns into a working Akka project — agents, workflows, governance, UI, and tests — on your machine.

Run any sample:

  1. Install Akka Specify for Claude Code (one-time; see the install guide).
  2. git clone https://github.com/TylerJewell/akka-blueprints.git
  3. cd akka-blueprints/<sample-folder>
  4. Open Claude Code and run /akka:specify @SPEC.md
